# Custom Commands

## /test
Run the Maven test suite for the Commons project. Executes `mvn test` from the project root, which runs all tests across all modules.

```bash
mvn test
```

## /test-util
Run tests only for the Util module.

```bash
mvn test -pl Util
```

## /test-graph
Run tests only for the Graph module.

```bash
mvn test -pl Graph
```

## /test-reflection
Run tests only for the Reflection module.

```bash
mvn test -pl Reflection
```
